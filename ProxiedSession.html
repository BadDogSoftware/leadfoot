Module: leadfoot/ProxiedSession

leadfoot/ProxiedSession

new require("leadfoot/ProxiedSession")(sessionId, server, capabilities)

A ProxiedSession object represents a WebDriver session that interacts with the Intern instrumenting proxy. It collects code instrumentation data from pages and converts local filesystem paths into URLs for use with module:leadfoot/Session#get.
Parameters:
Name Type Description
sessionId string The ID of the session, as provided by the remote.
server module:leadfoot/Server The server that the session belongs to.
capabilities Object A map of bugs and features that the remote environment exposes.
